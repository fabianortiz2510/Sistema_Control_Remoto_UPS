<h1 align="center"> Sistema_Control_Remoto_UPS </h1>

  
El presente proyecto presenta un sistema remoto de monitoreo de infraestructura crítica que permite obtener datos de los parámetros de funcionamiento de las Unidades de Potencia Ininterrumpida (UPS).
Este sistema facilita la supervisión en tiempo real de variables como el estado operativo, nivel de carga, temperatura y voltaje, permitiendo una gestión eficiente y preventiva del sistema eléctrico.
La solución se basa en la integración de comunicación remota mediante protocolos industriales, almacenamiento estructurado de datos y visualización a través de dashboards interactivos, optimizando la toma de decisiones y el mantenimiento de los equipos.
